# 英语学习助手 (EngMemo)

一个功能强大的英语单词学习应用，支持多单词本管理、智能搜索、语音播放和测验功能。

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- npm 或 yarn

### 安装和启动

1. **克隆或下载项目**
   ```bash
   # 如果是从Git仓库克隆
   git clone <repository-url>
   cd engmemo
   
   # 或者直接下载解压到文件夹
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **访问应用**
   - 打开浏览器访问：http://localhost:5173
   - 应用会自动打开默认浏览器

### 构建生产版本
```bash
npm run build
```

## 📚 功能特性

### 1. 多单词本管理
- **多单词本支持**：创建多个独立的单词本
- **自定义命名**：为每个单词本设置名称和描述
- **快速切换**：在单词本之间快速切换
- **独立管理**：每个单词本有独立的单词集合

### 2. 智能单词管理
- **一键获取信息**：自动获取IPA、中文释义、英文释义
- **美式/英式发音**：支持美式和英式发音选择
- **语音播放**：内置TTS和真实音频播放
- **唯一性检查**：防止重复添加相同单词
- **标签系统**：为单词添加自定义标签

### 3. 高级搜索功能
- **多范围搜索**：支持英文、中文、标签、全部范围搜索
- **实时过滤**：输入即时显示搜索结果
- **搜索统计**：显示找到的单词数量
- **一键清空**：快速清空搜索条件

### 4. 智能测验系统
- **多种模式**：中译英、英译中
- **智能排序**：随机、顺序、低熟练度优先
- **熟练度跟踪**：自动记录学习进度
- **即时反馈**：实时显示答题结果

### 5. 数据管理
- **本地存储**：数据保存在浏览器本地
- **导入导出**：支持JSON格式数据备份
- **设置持久化**：个人设置自动保存

## 🎯 使用指南

### 首次使用

1. **创建单词本**
   - 进入"单词本"页面
   - 点击"📚 管理单词本"按钮
   - 输入单词本名称和描述
   - 点击"创建"按钮

2. **添加单词**
   - 在单词本页面点击"新增单词"
   - 输入英文单词
   - 点击"获取中文"或"获取所有"自动填充信息
   - 点击"保存"完成添加

### 日常使用

#### 单词管理
- **查看单词**：在单词本页面浏览所有单词
- **编辑单词**：点击"编辑"按钮修改单词信息
- **删除单词**：点击"删除"按钮移除单词
- **播放发音**：点击发音按钮听单词读音

#### 搜索单词
- **基本搜索**：在搜索框输入关键词
- **范围搜索**：选择搜索范围（全部/英文/中文/标签）
- **清空搜索**：点击搜索框右侧的 ✕ 按钮

#### 开始测验
- **选择模式**：在设置中选择测验模式
- **开始测验**：进入"测验"页面开始学习
- **查看结果**：系统会显示答题正确性

#### 管理单词本
- **创建新单词本**：为不同主题创建独立单词本
- **切换单词本**：在单词本管理器中切换
- **编辑单词本**：修改单词本名称和描述
- **删除单词本**：删除不需要的单词本

## ⚙️ 设置说明

### 语音设置
- **语速调节**：0.5x - 1.5x 播放速度
- **重复次数**：1-10次重复播放
- **语音选择**：选择系统可用语音
- **音标显示**：开启/关闭IPA音标显示

### 测验设置
- **测验模式**：
  - 中译英：显示中文，输入英文
  - 英译中：显示英文，输入中文
- **排序方式**：
  - 随机：随机选择单词
  - 顺序：按添加顺序
  - 低熟练度优先：优先测验不熟悉的单词

## 🐛 常见问题

### Q: 为什么获取不到中文翻译？
A: 请检查网络连接，系统会尝试多个翻译源。对于常用单词，系统有内置的准确翻译。

### Q: 语音播放不工作？
A: 请确保浏览器支持Web Speech API，并允许音频播放权限。

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地存储中，清除浏览器数据会丢失。建议定期使用导入导出功能备份数据。

### Q: 如何备份数据？
A: 进入"导入导出"页面，点击"导出数据"下载JSON文件。
